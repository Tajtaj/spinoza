Commands | Dump Closest Trends

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX dumpClosestTrends -lat 0.0 -long 0.0 -o ClosestTrends.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpClosestTrends -lat 0.0 -long 0.0 -o ClosestTrends.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-lat:

Latitude of place (e.g 51.5072 for London).

-long:

Longitude of place (e.g -0.1275 for London).

-o:

Name of output file to which the result will be dumped.

Description:

This command dumps locations for which the twitter has trending topic information that is closest to the specified location, to the provided output file.
