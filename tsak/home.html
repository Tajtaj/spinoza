Welcome to the twitter-swiss-army-knife!

The purpose of this project is to create a simple command line utility that allows a user to interact with Twitter's public API. TSAK is an open source application that enables end users use twitter APIs without going to in-depth twitter technicalities. All the end user has to do is to provide twitter credentials (App + User keys) and action query that depend upon what the user wants to access and let the TSAK does its magic.
