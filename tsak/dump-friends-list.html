Dump by Twitter Screen Name:

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX dumpFriendsList -uname screenname -o friendlist.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpFriendsList -uname screenname -o friendlist.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-uname:

The screen_name of the user for whom to return results for.

-o:

Name of the output file to which the Result Users will be dumped.

Description:

This command dumps a collection of users that the specified user is following (also known as their “friends”). The number of friend are subjected to Twitter rate limits.
