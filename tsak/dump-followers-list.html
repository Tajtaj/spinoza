Dump by Twitter Screen Name:

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX dumpFollowersList -uname screenname -o followers.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpFollowersList -uname screenname -o followers.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-uname:

The screen_name of the user for whom to return results for.

-o:

Name of the output file to which the Result Users will be dumped.

Description:

This command dumps a collection of users that the specified user is followed by (otherwise known as their “followers”).




Dump by Twitter ID:

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX dumpFollowersList -uid 010101 -o followers.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpFollowersList -uid 010101 -o followers.txt

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-uid:

The ID of the user for whom to return results for.

-o:

Name of the output file to which the Result Users will be dumped.

Description:

This command dumps a collection of users that the specified user is followed by (otherwise known as their “followers”).The number of Users to be retrieved, are subjected to Twitter rate limits.
