Commands | Dump User List Members

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX dumpUserListMembers -lid 02020202 -o output.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpUserListMembers -lid 02020202 -o output.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-lid:

The numerical id of the list.

-o:

Name of the output file to which the List members will be dumped.

Description:

This command dumps all the members of the specified list. Private list members will only be shown if the authenticated user owns the specified list.
