Commands | Dump Place Trends

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX dumpPlaceTrends -woeid 010101 -o trends.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpPlaceTrends -woeid 010101 -o trends.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-woeid :

The Yahoo! Where On Earth ID of the location.

-o:

Name of the output file to which the result trends will be dumped.

Description:

This command dumps top 10 trending topics for the specific WOEID to the given output file, if trending information is available for it.
