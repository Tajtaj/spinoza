Commands | Dump retweeters by Status id

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX dumpRetweeters -sid 01010101 -o Retweeters.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpRetweeters -sid 01010101 -o Retweeters.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-sid:

Status ID of the desired status.

-o:

Name of the output file to which the Retweeters of the Specified Tweet will be dumped.

Description:

This command dumps retweeters of the specified Tweet to the given output file, as per provided status-id. The number of dumped retweeters are subjected to Twitter rate limits.
