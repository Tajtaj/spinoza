Commands | Dump Block Users

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX dumpBlockList -o blocklists.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpBlockList -o blocklists.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-o:

Name of the output file to which list of blocked users will be dumped.

Description:

This command dumps the blocked users list of twitter authenticated user to the provided output file. The number of blocked users retrieved are subjected to Twitter rate limits.
