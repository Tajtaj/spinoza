Commands | Search Place

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX searchPlace -lat 51.5072 -long 0.1275 -o places.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak searchPlace -lat 51.5072 -long 0.1275 -o places.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-lat:

Latitude of the place, which is to be searched.

-long:

Longitude of the place, which is to be searched.

-o:

Name of the output file to which the result will be dumped.

Description:

This command will dump all the information about the required place, of which latitude and longitude are provided.
