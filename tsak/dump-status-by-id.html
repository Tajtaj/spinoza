Commands | Dump status by id

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX dumpStatus -sid 01010101 -o status.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpStatus -sid 01010101 -o status.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-sid:

Status ID of required status.

-o:

Name of the output file to which the Twitter status will be dumped.

Description:

Based on provided credentials, this command dumps a twitter Status to a file as per provided status-id.
