Commands | Dump User List Subscribers

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX dumpUserListSubscribers -lid 02020202 -o subscribers.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpUserListSubscribers -lid 02020202 -o subscribers.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-lid:

Numerical twitter ID of a list.

-o:

Name of the given output file to which the List Subcribers will be dumped.

Description:

This command dumps all subscribers of the specified list to the given output file. Private list subscribers will only be shown if the authenticated user owns the specified list. The number of retrieved twitter lists are subjected to Twitter rate limits.
