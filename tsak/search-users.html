Commands | Search Users

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX searchUsers -keywords="Key Word" -o Users.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpBlockList -o blocklists.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-keywords:

Search Twitter users whose names or screen names contain these keywords.

-o:

Name of the output file to which the Result Users will be dumped.

Description:

This command dumps the users list whose names or screen names matches the keywords.
