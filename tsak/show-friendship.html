Commands | Show Friendship

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXXX -consumerSecret=XXXXX -accessToken=XXXXX -accessSecret=XXXXX showFriendShip -suser ScreenName -tuser ScreenName -o friendship.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak showFriendShip -suser ScreenName -tuser ScreenName -o friendship.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-suser:

ScreenName or ID of the source user.

-tuser:

ScreenName or ID of the Target user.

-o:

Name of the output file to which the result relation will be dumped.

Description:

This command dumps the relationship properties between the source user and target user, to the given ouput file. Either twitter screen names or IDs will be needed for both source and target user.
