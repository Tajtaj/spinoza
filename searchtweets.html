Search by Keywords:

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXX -consumerSecret=XXXX -accessToken=XXXX -accessSecret=XXXX dumpTweets -keywords "KEY words" -o output.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpTweets -keywords "KEY words" -o output.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-keywords:

Search tweets containing these keywords

-o:

Name of the output file to which the searched Tweets will be dumped.

Description:

Based on provided Credentials, this command dumps searched Tweets containing the provided keywords to a file. The number of dumped Tweets are subjected to Twitter rate limits and cannot exceeds 2700/15min.
