Dump by Twitter Screen Name:

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXX -consumerSecret=XXXX -accessToken=XXXX -accessSecret=XXXX dumpFriendIDs -uname ScreenName -o output.csv

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpFriendIDs -uname ScreenName -o output.csv

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-uname:

Target User Twitter ScreenName.

-o:

Name of the output file to which the target user follower will be dumped.

Description:

Based on provided Credentials, this command dumps a twitter user's friends ids, ScreenName, followers count, and friends count to a file. The number of friends id, ScreenName, followers count, friends count are subjected to Twitter rate limits and cannot exceeds 75000/15min.
